# Decision Log

> 此文件记录关键决策，便于跨会话追溯。
> 格式：每条记录包含 Date(UTC) · Scope · Problem · Decision · Location · Impact。
> 详见 `.cursor/rules/memory-bank-workflow.mdx`。
